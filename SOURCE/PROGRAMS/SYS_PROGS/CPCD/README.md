# CPCD — Copy CD-ROM (ISO9660) to Disk Utility

*CPCD* is a simple utility for copying files from a CD-ROM (ISO9660) to the hard drive. It is designed to work with the FAT32 file system and can handle large files efficiently.

## Usage

```
CPCD.BIN <SOURCE> <DESTINATION>

SOURCE: The path inside the CD-ROM ISO image (e.g., /ATOS/CPCD.BIN)
DESTINATION: The path on the hard drive where the file should be copied (e.g., /CPCD-COPY.BIN)
```
