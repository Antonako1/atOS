# ATA_PIIX3

ATA PIIX3 driver for RTOS kernel.

This driver provides support for ATA devices connected via the PIIX3 chipset, enabling communication between the operating system and ATA-compliant hardware for data transfer and device management.

## Helpful information

The whole ATA system can support 4 drives, 2 per channel (Primary and Secondary). Each channel can have a master and a slave drive.
